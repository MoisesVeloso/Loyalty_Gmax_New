# Gmax Loyalty System

## Description
The **Gmax Loyalty System** is a web-based application designed to track and manage loyalty points for customers. It enables businesses to manage customer transactions and reward their loyalty with a point system.

## Features
- **Customer Registration**: A page where customers can register their details to create a profile.
- **Dashboard**: A user-friendly dashboard to view customer data and their current loyalty points balance.
- **Transaction Modal**: An interface to add, update, and track transactions, which will affect the customer’s loyalty points.

## Installation

### Prerequisites
- Web browser (Chrome, Firefox, etc.)
- Node.js (if the backend uses Node.js)
- Database (e.g., MySQL, MongoDB) configured.
