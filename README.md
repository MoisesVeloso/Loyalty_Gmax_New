# Gmax Loyalty System

## Description
The **Gmax Loyalty System** is a simple web-based application designed to track and manage loyalty points for customers.

**Note**: This system is designed to run in a local development environment and is optimized for running on your local machine using **XAMPP** for the database setup.

## Features
- **Customer Registration**: A page where customers can register their details to create a profile.
- **Dashboard**: A user-friendly dashboard to view customer data and their current loyalty points balance.
- **Transaction Modal**: An interface to add, update, and track transactions, which will affect the customer’s loyalty points.

## Installation

### Prerequisites
- Web browser (Chrome, Firefox, etc.)
- Node.js
- Xampp (for SQL)

**Before running the system locally, you need to install the following tools:**

1. **Git (Optional can use windows command terminal)**  - To clone the repository to your local machine.
   - Download Git from [here](https://git-scm.com/downloads){:target="_blank"} and follow the installation instructions.

2. **Node.js** - To run the backend server.
   - Download Node.js from [here](https://nodejs.org/en/download/){:target="_blank"}
   - Make sure to install the **LTS** version.

3. **XAMPP** - To run MySQL locally for the database.
   - Download XAMPP from [here](https://www.apachefriends.org/index.html){:target="_blank"} and install it.
