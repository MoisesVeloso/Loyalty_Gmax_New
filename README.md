# Gmax Loyalty System

## Description
The **Gmax Loyalty System** is a simple web-based application designed to track and manage loyalty points for customers.

## Features
- **Customer Registration**: A page where customers can register their details to create a profile.
- **Dashboard**: A user-friendly dashboard to view customer data and their current loyalty points balance.
- **Transaction Modal**: An interface to add, update, and track transactions, which will affect the customer’s loyalty points.

## Installation

### Prerequisites
- Web browser (Chrome, Firefox, etc.)
- Node.js
- Xampp (for SQL)
